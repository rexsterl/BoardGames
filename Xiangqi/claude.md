# 2D Xiangqi (Chinese Chess) Game

## Project Goal
Build a playable 2D chinese chess game where a human player can compete against an AI opponent.

## Tech Stack
- Python 3.x
- Pygame for graphics and UI
- Chinese Chess logic library (python-chess recommended for move validation)

## Key Features
- Visual Chinese Chess board with piece rendering
- Legal move validation
- Player vs Computer gameplay
- Basic AI opponent (minimax algorithm with alpha-beta pruning)
- Move highlighting and piece selection
- Game state display (check, checkmate, stalemate)

## Project Structure
- Main game loop and rendering
- Chinese Chess board state management
- AI engine for computer moves
- UI for piece interaction
